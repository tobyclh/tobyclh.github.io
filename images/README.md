# Background Images

To add tiled background images to each page, place your images in this directory with the following filenames:

- `bg-top.jpg` - Background for the Top/Home page (index.html)
- `bg-publications.jpg` - Background for the Publications page
- `bg-works.jpg` - Background for the Works page
- `bg-educations.jpg` - Background for the Educations page
- `bg-contacts.jpg` - Background for the Contacts page

## Template Image

You can use this image as a template:
https://images.prestigeonline.com/wp-content/uploads/sites/3/2023/04/04115245/abe-hiroshi-excellent-asian-cinema-award.jpg

The images will be tiled (repeated) across the background of each page.

## Image Requirements

- Format: JPG, PNG, or GIF
- Recommended size: 200-400px for optimal tiling effect
- Keep file sizes small to maintain fast loading times
